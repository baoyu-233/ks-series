# ItemEditor 玩家使用指南

> [English](GUIDE_PLAYER.en.md) | 中文

> 你是普通玩家，可以使用网页设计器和武器精炼功能。

## 一、网页物品设计器（`/design`）

在聊天框输入 `/design`，会得到一个可点击的链接。

### 打开链接后

浏览器中打开，你会看到：

**左侧**：
- 📦 **材质选择器** — 搜索或按分类浏览所有物品材质
- 🎭 **特殊符号面板** — 展开后 6 类符号，点击插入

**右侧**：
- 👁️ **物品预览** — 实时显示名称和 Lore 的实际效果
- ⚙️ **物品配置** — 编辑名称、Lore、附魔
- 📋 **模板管理** — 保存/导入/加载模板

### 编辑物品名称
- 直接在"物品名称"输入框中输入
- 支持 `&` 颜色码：`&6神剑` → 金色"神剑"
- 点击输入框会出现**格式化工具栏**：
  - 16 个颜色方块 → 点击插入颜色码
  - **B** 粗体 / *I* 斜体 / <u>U</u> 下划线
  - **选中文本**再点格式按钮 → 自动包裹 + 末尾恢复

### 编辑 Lore（描述）
- 在输入框中填写内容，点击 **+ 添加**
- 每条 Lore 可独立使用颜色码和格式码
- 上下箭头调整顺序，✕ 删除

### 添加附魔
- 从下拉框选择附魔
- 用 **-** / **+** 按钮调整等级
- 达到附魔自然上限时显示「已达上限」提示
- 冲突附魔会显示 ⚠️ 警告

### 预览面板
在"物品配置"上方，**实时显示**名称 + Lore 的最终效果：
- 颜色码会自动渲染为对应颜色
- `&l` 显示为粗体、`&o` 斜体、`&n` 下划线
- Emoji 和特殊符号直接显示
- 修改名称或 Lore 立刻刷新

## 二、模板系统

设计完成后，可以保存为模板分享给其他玩家，或自己以后复用。

### 保存模板
1. （可选）填写模板名称，如"神剑"
2. 点击 **💾 保存玩家模板**
3. 得到模板码，如 `pl-a1b2c3d4`
4. 复制模板码发送到群里分享

### 加载模板
- **网页中**：在"导入模板"框输入模板码 → 📥 导入预览 → 可修改后重新保存
- **游戏中**：输入 `/design load pl-a1b2c3d4`

### 我的模板
点击 📂 **我的模板** 查看所有保存过的模板，可加载或删除。

### 模板码说明
- `pl-` 开头 = 玩家模板（你可加载）
- `op-` 开头 = 管理员模板（你**无法加载**，会提示无权）

## 三、武器精炼（`/refine`）

消耗**兑换券**对武器进行强化。

### 使用方法
1. 手持要精炼的武器
2. 输入 `/refine`
3. 在 GUI 中编辑名称/Lore/附魔
4. 点击 **确认** → 消耗兑换券 → 武器精炼完成
5. 点击 **取消** 或关闭 GUI → 武器原样归还

### 兑换券
服务器管理员会告诉你兑换券是什么物品。常见形式：
- 特定名称的物品（如"精炼券"）
- MythicMobs 物品

### 精炼限制
- 附魔等级最高 **10** 级（由服主配置）
- 不可突破原版附魔上限
- 不包含 FotiaEnchantment 附魔

### 安全机制
- 编辑期间物品从背包取出锁定
- 不小心关闭 GUI → 物品自动归还
- 断线 → 物品自动归还

## 四、颜色码速查表

在名称和 Lore 中使用 `&` + 以下代码：

| 代码 | 颜色 | 代码 | 颜色 |
|:--:|------|:--:|------|
| `&0` | 黑色 | `&8` | 深灰 |
| `&1` | 深蓝 | `&9` | 蓝色 |
| `&2` | 深绿 | `&a` | 绿色 |
| `&3` | 青色 | `&b` | 天蓝 |
| `&4` | 深红 | `&c` | 红色 |
| `&5` | 紫色 | `&d` | 粉色 |
| `&6` | 金色 | `&e` | 黄色 |
| `&7` | 灰色 | `&f` | 白色 |

| 代码 | 效果 |
|:--:|------|
| `&l` | **粗体** |
| `&o` | *斜体* |
| `&n` | <u>下划线</u> |
| `&m` | ~~删除线~~ |
| `&k` | 乱码闪烁 |
| `&r` | 重置所有格式 |

## 五、提示和技巧

1. **善用预览面板** — 边改边看效果，不用反复进游戏测试
2. **先给模板起名** — 模板多了方便管理
3. **导入 + 修改 = 新模板** — 看到别人的好模板？导入后改一改，保存为自己的版本
4. **格式码叠加** — `&6&l神剑` = 金色 + 粗体
5. **颜色码会重置格式** — 每个颜色码 `&0`~`&f` 会自动清除之前的粗体/斜体等格式
6. **用 `&r` 恢复** — 如果只想给部分文字加格式，后面加 `&r` 恢复

## 六、常见问题

**Q: 为什么我保存的模板别人加载了没有属性修饰符？**
A: 属性修饰符是管理员专属功能，玩家模板不包含。

**Q: 为什么我加载模板提示"无权"？**
A: 该模板是管理员专属（`op-` 开头），玩家无法加载。

**Q: FE 附魔是什么？我能用吗？**
A: FotiaEnchantment 是服务器的自定义附魔系统。玩家在网页编辑器里看不到 FE 附魔选项——这是管理员专属功能。

**Q: 精炼券不够了怎么办？**
A: 联系服主获取更多兑换券。

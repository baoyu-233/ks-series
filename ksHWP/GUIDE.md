# 🗺️ ksHWP 世界地图 — 玩家使用指南

> [English](GUIDE.en.md) | 中文

## 这是什么？

ksHWP 是本服的**公共社区地图**。它会把 Minecraft 世界的地形渲染成网页地图，任何人都可以在浏览器里查看。更重要的是，这是**大家一起绘制**的地图——你去过的地方会自动标记在地图上，后来的人就能看到。

## 如何打开？

### 方法一：游戏内命令（推荐）
在游戏里输入：
```
/map
```
聊天栏会弹出一个链接，**点击就能在浏览器里打开**。这个链接包含了你的身份信息，推荐用这个方式。

### 方法二：直接访问
浏览器打开 `http://<服务器IP>:8123/kSHWP`，但不带 token 的话无法添加备注和更新区域。

## 地图操作

| 操作 | 方式 |
|------|------|
| **平移** | 鼠标左键拖拽 |
| **缩放** | 鼠标滚轮 / 右侧 +/- 按钮 |
| **选点** | 右键点击地图 |
| **框选区域** | Shift + 鼠标左键拖拽 |
| **切换世界/维度** | 左侧边栏点击世界名 |

## 世界与维度

左侧边栏列出了所有可用的世界。点击即可切换。每个世界有独立的地图，**切换世界不会丢失之前加载的内容**。

- 🟢 **主世界 (world)** — 主地图
- 🔴 **地狱 (world_nether)** — 下届
- 🟣 **末地 (world_the_end)** — 末地

## 地图颜色说明

| 颜色 | 含义 |
|------|------|
| 🟩 绿色 | 草地 |
| 🟦 蓝色 | 水域 |
| ⬜ 灰色 | 石头/建筑 |
| 🟥 **红色 + "未探索"** | 还没有玩家去过的地方，等你来发现！ |

> 💡 **公共地图机制**：任何玩家去过的地方都会被渲染并永久保存。你去探索，后来的人就能看到。大家一起画地图！

## 地图备注 📌

在地图上标注重要位置，方便所有人查看。

### 添加备注
1. **右键** 地图上的位置 → 选 "★ 点标注"
2. 在左侧边栏填写备注标题、补充说明、类型、颜色
3. 点击 "添加备注"

也支持**框选区域**（Shift+拖拽）来标注城镇、建筑群等大范围区域。

### 查看备注
左侧边栏"地图备注"列表，点击任一条目会跳转到对应位置。

### 搜索备注
使用搜索框输入关键词，支持按类型筛选。

### 删除自己的备注
你创建的备注右侧有红色 **删除** 按钮，点击即可删除。

### 备注类型
- 点标注 — 单个位置
- 城区规划 — 城镇区域
- 地标 — 自然地标
- 建筑 — 建筑物
- 农场/牧场 — 农业区
- 道路 — 交通路线
- 矿场 — 矿区

## 更新区域 🔄

如果你在某个区域建造了东西，想让地图反映最新状态：

1. **右键** 或 **Shift+拖拽** 选中区域
2. 在弹出菜单中点击 "🔄 更新此区块"
3. 地图会重新渲染该区域

> 注意：需要从游戏内 `/map` 获取的链接才能使用此功能。

## 隐藏模式 👁

如果你不希望自己的位置在地图上显示：
```
/map hidden
```
再次输入此命令可取消隐藏。需要 `kshwp.hidden` 权限（默认 OP）。

## 在线玩家

左侧边栏会实时显示在线玩家及其所在维度。地图上绿色圆点表示玩家位置（隐藏模式的玩家不会显示）。

## 常见问题

**Q: 为什么有些地方是红色的"未探索"？**
A: 因为还没有玩家去过那里。飞过去探索一下，地图就会自动渲染。

**Q: 缩小地图后之前渲染的区域不见了？**
A: 不会的。缩小后会自动从更详细的图块合成，只是可能有短暂加载时间。

**Q: 我能看到别人的备注吗？**
A: 可以看到公开备注和服务器标注。每个人的私有备注只有自己能看到。

**Q: 地图数据会丢失吗？**
A: 不会。所有渲染过的地形都保存在服务器磁盘上，重启也不会丢失。

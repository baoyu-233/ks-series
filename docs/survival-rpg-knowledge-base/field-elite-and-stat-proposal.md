# 寒地精英与首批战术装备方案

状态：重做后的待实装规格。此前“略高于铁装、无主动技能”的方案废弃；本文件是首批
装备、野外精英和技能的唯一数值来源，尚未写入服务器配置。

## 强度基线

服务器的玩家已拥有完整生存进度，首批 RPG 配装必须建立在满附魔下界合金之上，而不是
要求玩家为了 RPG 降级装备。原版护甲、保护 IV、耐久、药水和食物仍然有效；RPG 武器与
两戒一护符在其上提供新的战术能力。

- 标准武器的 30 秒单目标平均 DPS 必须达到同操作下锋利 V 下界合金剑的 `110%` 以上。
- 精制武器在无机制窗口时达到该基线的 `125-140%`；成功完成团队机制后才允许短时达到
  `160%` 左右，不能常驻碾压。
- 生存构筑在完整下界合金甲之上额外提供生命、护甲、韧性与救命技能，足以正面承担精英
  攻击，但不能吃掉首领的连续未躲避技能。
- 所有数值须以当前 MMOItems 的实际属性叠加方式和实测伤害为准。YAML 中的
  `attack-damage` 是物品攻击点数，不能仅凭配置阅读判断最终伤害。

## 霜原幼龙

复用已被 `Frostbound_Conductor` 实际加载的 `pet_ice_dragon` 模型，ID 为
`Frostwaste_Whelp`。它是寒地常规野外精英，不是缩血首领：无首领条、无威胁表、无冰牢、
无白雾、无阶段机制。

| 项目 | 初始值 |
| --- | --- |
| 基底实体 / 生命 / 伤害 / 护甲 | WOLF / `360` / `14` / `12` |
| 移速 / 跟随范围 / 抗击退 | `0.29` / `24` / `0.55` |
| 出现 | 寒冷群系，夜间或降雪；每个区域同时最多 1 只 |
| 刷新 | 每区域 `10-15` 分钟随机一次，48 格内有玩家才生成 |
| 消失 | 脱战 90 秒后移除，不持久化 |
| 技能 | `寒息喷吐`：2 秒雪雾前摇，8 格锥形造成 `7` 点伤害和缓慢 II 2 秒，12 秒冷却 |

先按现有 `model{mid=pet_ice_dragon}` 测试默认尺寸和碰撞体，再决定缩放参数；不能猜测
当前 ModelEngine R4.0.9 的缩放语法。

## 武器投放

推荐混合制。精英稳定掉可交易材料，低概率掉标准武器，精制品由生存产线制造。这样首把
强力武器可以通过探索获得，但重复刷怪不会取代制作和市场。

| 掉落 | 概率 / 数量 | 说明 |
| --- | --- | --- |
| `MATERIAL:KS_FIELD_SCRAP` 野外废料 | 100%，`4-6` | 接入既有兑换链 |
| `MATERIAL:KS_FROST_SCALE` 霜鳞 | 70%，`2-4` | 标准与精制武器的寒地材料 |
| `MATERIAL:KS_WHELP_CORE` 幼龙核心 | 18%，`1` | 精制武器组件，不能由自动机器生成 |
| 野外军备匣 | 1.5%，`1` | 一次只给一把标准武器：长剑 45%、短弓 30%、战锤 25% |

军备匣的开启与个人掉落资格必须由未来 `ks-RPG` 掉落服务完成，不能用客户端背包操作或
MythicMobs 定时器绕过服务端校验。资格为死亡前 20 秒内造成或承受过该怪伤害的参与者；
每只怪只结算一份军备匣，不按人数复制。标准武器另有制作路径，精制武器必须使用霜鳞、
幼龙核心、现有合金/线圈和金币。

可选投放方式：

| 方案 | 结果 |
| --- | --- |
| 成品掉落 | 最直观，但长期会让刷点决定装备供应 |
| 纯制作 | 经济最稳，但第一次探索缺少惊喜 |
| 混合制，采用 | 低概率标准成品 + 稳定材料 + 精制制作，兼顾探索与经济 |

## 武器与武器技

所有武器技由 `ks-RPG` 统一处理，触发为“潜行 + 右键主手武器”。不使用 MMOCore 职业键位，
不允许同一技能在多个插件各自结算。每次施放都在服务器线程验证武器 ID、冷却、目标距离、
副本锁装和玩家在线状态。

| ID / 名称 | 品质与来源 | 静态属性 | 武器技 |
| --- | --- | --- | --- |
| `SWORD:KS_TRAILBLAZER_BLADE` 开拓者长剑 | `KS_STANDARD`；军备匣或制作 | 攻击 `15.0`；攻速 `1.85`；暴击 `6%`；暴伤 `18%`；武器伤害 `5%` | `开路斩`：前方 5 格扇形造成 `9 + 75%` 攻击伤害，命中后自身急迫 I 与速度 I 2 秒；冷却 7 秒。 |
| `BOW:KS_ICEFALL_SHORTBOW` 冰痕短弓 | `KS_STANDARD`；军备匣或制作 | 攻击 `15.0`；暴击 `10%`；暴伤 `25%`；投射物伤害 `10%` | `猎印箭`：下一箭命中为目标施加“猎印”5 秒，队友对其伤害 `+10%`；同一目标仅保留一枚猎印；冷却 14 秒。 |
| `HAMMER:KS_SALVAGER_MAUL` 拾荒战锤 | `KS_STANDARD`；军备匣或制作 | 攻击 `20.0`；攻速 `0.95`；暴击 `2%`；暴伤 `32%`；武器伤害 `8%`；移速 `-0.006` | `崩岩`：3.5 格震击，造成 `11 + 70%` 攻击伤害；普通怪眩晕 0.8 秒，首领改为 4 秒 `+6%` 受伤；冷却 12 秒。 |
| `SPEAR:KS_RELAY_PIKE` 接力长矛 | `KS_REFINED`；霜鳞、精制合金、导能线圈、金币 | 攻击 `18.0`；攻速 `1.55`；暴击 `10%`；暴伤 `28%`；武器伤害 `8%` | `接力突刺`：向准星方向突进至多 4.5 格，命中造成 `10 + 70%` 攻击伤害；下一名不同玩家在 3 秒内命中同目标时额外造成 `12%` 武器伤害；冷却 10 秒。 |
| `HAMMER:KS_QUARRY_MAUL` 采掘重锤 | `KS_REFINED`；霜鳞、幼龙核心、精制合金、金币 | 攻击 `24.0`；攻速 `0.78`；暴击 `5%`；暴伤 `45%`；武器伤害 `12%`；移速 `-0.014` | `断层`：3 格重击造成 `15 + 100%` 攻击伤害。若目标正处于团队机制成功后的易伤期，额外造成 `18%` 武器伤害；冷却 16 秒。 |
| `BOW:KS_WINDWARD_BOW` 望风短弓 | `KS_REFINED`；霜鳞、精制合金、导能线圈、金币 | 攻击 `18.0`；暴击 `14%`；暴伤 `32%`；投射物伤害 `15%` | `风眼`：发射穿透一名目标的疾风箭，造成 `12 + 80%` 攻击伤害并施加猎印 6 秒；若队友先触发猎印，弓手返还 5 秒本技能冷却；冷却 18 秒。 |

标准级长剑、短弓和战锤均已达到满附魔原版的战斗基线；精制长矛、重锤和短弓依靠技能
协作进一步上升。第一批暂不提供精制长剑，避免长剑同时拥有最高稳定性和最高上限。

## 饰品、护符与技能

仍然使用两戒一护符。输出戒指 `势能`、`聚焦`、`共鸣` 每次副本最多装备一枚；同 ID 戒指
不能重复。护符技同样使用潜行右键，但与武器技共享 1 秒公共施放锁，避免一次输入连放两技。

| ID / 名称 | 静态属性 | 效果 |
| --- | --- | --- |
| `RING:KS_RING_OF_BASTION` 堡垒之戒 | 生命 `+6`；护甲 `+2`；韧性 `+1` | `余烬守护`：生命低于 35% 时获 6 点吸收 4 秒，60 秒冷却。 |
| `RING:KS_RING_OF_RESOLVE` 决意之戒 | 生命 `+8`；护甲 `+3`；韧性 `+1.5` | `不屈`：2 秒内累计承受 12 点伤害后获得抗性 I 3 秒，30 秒冷却。 |
| `RING:KS_RING_OF_MOMENTUM` 势能之戒 | 攻速 `+0.18`；武器伤害 `+5%` | 武器技命中后，下一次普攻额外 `+8%` 武器伤害；每次技能仅触发一次。 |
| `RING:KS_RING_OF_FOCUS` 聚焦之戒 | 暴击 `+8%`；暴伤 `+35%` | 对被猎印或处于首领易伤期的目标，暴击率再 `+4%`。 |
| `RING:KS_RING_OF_STRIDE` 行旅之戒 | 移速 `+0.015`；抗击退 `+0.10` | 成功躲开霜原幼龙寒息或首领技能伤害后，获得速度 II 2 秒，10 秒冷却。 |
| `RING:KS_RING_OF_RESONANCE` 共鸣之戒 | 投射物伤害 `+15%`；暴击 `+4%` | 自己施加的猎印被队友触发时，返还 3 秒武器技冷却。 |
| `TALISMAN:KS_PIONEER_TALISMAN` 先驱护符 | 生命 `+6`；护甲 `+1` | `远征意志`：自身或 8 格内队友首次低于 40% 生命时，获得再生 I 4 秒；每名队员每 60 秒一次。 |
| `TALISMAN:KS_ANCHOR_TALISMAN` 定锚护符 | 生命 `+8`；护甲 `+2`；抗击退 `+0.18`；移速 `-0.006` | `定锚`：3 秒内受伤 `-25%` 且免疫击退，不能冲刺；45 秒冷却。 |
| `TALISMAN:KS_SIGNAL_TALISMAN` 讯号护符 | 移速 `+0.010`；攻速 `+0.10`；冷却缩减 `+6%` | `集结讯号`：自身周围 8 格队友获得速度 I、急迫 I 6 秒；35 秒冷却。 |
| `TALISMAN:KS_HUNTER_TALISMAN` 猎手护符 | 武器伤害 `+9%`；暴伤 `+18%`；生命 `-2` | `追猎`：对猎印或易伤目标的下一次武器技额外 `+12%` 武器伤害；每次标记只触发一次。 |

极限生存构筑 `决意 + 堡垒 + 定锚` 在完整下界合金甲上提供 `+22` 生命、`+7` 护甲、
`+2.5` 韧性和一次定锚减伤，能够承担精英；极限输出构筑仍只能从三枚输出戒指中选一枚，
并因猎手负生命、武器移速/攻速惩罚和公共施放锁保留明显代价。

## 首轮验收

1. 用同一名无职业加成的玩家对假人、霜原幼龙和霜缚主宰分别测试满附魔下界合金与每把
   新武器的 30 秒 DPS，记录普攻与技能伤害。
2. 验证每种主动技能只能由对应物品触发，切换武器、死亡、掉线、离开副本均清空冷却和标记。
3. 对四人队验证猎印、接力突刺、集结讯号和易伤期重锤不会重复叠层，也不会对非队友生效。
4. 监测 20 分钟四人连续战斗的 MSPT、实体数和 GC；不得做循环背包扫描、同步 SQL 或异步
   Bukkit 访问。
5. 只在以上通过后，把 `ks-RPG` 技能引擎、MMOItems 物品、MythicMobs 精英和掉落服务打成
   同一批部署，避免为每件装备分别重启。

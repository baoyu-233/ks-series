# Survival RPG 知识库

> [English](README.md) | 中文

本知识库是 ks-Series 生存优先、逐步增加 RPG 深度的产品和系统事实源，记录方向、系统边界和分阶段交付。

- `vision.md`：玩家承诺、支柱和明确不做的内容。
- `content-architecture.md`：生存、经济、成长、探索和战斗的连接方式。
- `dungeon-economy-integration.md`：可购买准备与战斗获得能力的边界。
- `foundation-catalog.md`：第一季材料、品质和配装限制。
- `first-wave-loadouts.md`：首个副本前置配装与锁定契约。
- `field-elite-and-stat-proposal.md`：寒地精英与首批装备属性提案。
- `second-third-wave-content.zh-CN.md`：第二/第三波 ID 与内容包、三个待验收 Boss 包、赛季运行基础、
  契约冲突和仍未完成的实机接线。
- `roadmap.md`：阶段顺序和退出标准。
- `future-development.md`：长期生存到 RPG 的发展方向、专精边界与发布门槛。
- `decision-log.md`：已接受决策和开放问题。

Boss 设计和实测记录仍由 `docs/boss-knowledge-base/` 单独维护。

当前交付状态不是“一键上线”：第一波和第二波 ks-RPG source 内容已经存在；赛季存储与领域基础已经实现，
但默认关闭；第三波和 2026-07-18 的三个 Boss 包仍是 staging 草案。任何包 ID、证明、门槛、奖励接点或
遭遇在视为生产可用前，都应先核对 `second-third-wave-content.zh-CN.md`。

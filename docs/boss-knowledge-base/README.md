# Boss Knowledge Base

This directory is the encounter-design and validation sub-library of
`docs/survival-rpg-knowledge-base/`. It owns durable MythicMobs records for ks-Series bosses.

- `platform.md`: verified plugin versions, integration boundaries, and supported mechanics.
- `frostbound-conductor.md`: current encounter specification and tuning intent.
- `playtest-log.md`: dated observations and changes made in response.

Update this knowledge base whenever a boss mechanism is tested, rejected, retuned, or given a new runtime dependency.

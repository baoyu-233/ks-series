⚙️ 全局波动设置

关闭波动后，所有物品价格立即冻结为"仅受真实供需压力影响"，不再自动游走。波动率上限决定价格能偏离基准价多少。刷新间隔改了立即热重载，不用重启服务器。

👋 大手导向

给某个物品设置导向目标（-100%~+100%），价格会在接下来几轮刷新里逐渐、带随机波动地被拉向这个方向，不会瞬间跳价。设为 0 即清除导向。

🏛 任命元老 / 选举 📢 公告栏 →

元老院由 admin 任命元老；执政官由 admin 直接任命（不再自动）；保民官由全民投票按选举周期自动计票就任；骑士由选举引擎按企业资本排行自动维护（也可手动重算）。

⚠ 元老院能投票的前提：① 先任命足够元老（席位上限见「议会配置 → 元老席位」，太小会提示"元老院已满"）→ ② 直接任命执政官（提案必须由执政官/骑士发起，元老只能投票）→ ③ 执政官在玩家端 /player 提交提案并「发起表决」→ ④ 元老在玩家端投票。
若一直无法投票，多半是没有执政官（无人能发起提案）或元老席位太少。
